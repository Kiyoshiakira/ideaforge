Usage Guidelines

🎨 When to Use Each Color

  • Primary Purple: Primary actions, active states, important UI elements
  • Accent Gold: Call-to-action buttons, highlights, important notifications
  • Cyan: Links, informational elements, secondary actions
  • Success Green: Confirmations, success messages, completed states
  • Warning Orange: Warnings, caution messages, requires attention
  • Error Red: Errors, destructive actions, critical issues

♿ Accessibility Best Practices

  • All text on backgrounds meets WCAG AA (4.5:1) or AAA (7:1) contrast ratios
  • Never rely on color alone - use icons, labels, or patterns
  • Test with color blindness simulators
  • Ensure interactive elements have sufficient contrast in all states
  • Use semantic HTML with proper ARIA labels

💡 Implementation Tips

  • Use CSS custom properties for easy theme switching
  • Apply smooth transitions (0.3s) for theme changes
  • Save user preference in localStorage
  • Respect system preference with prefers-color-scheme
  • Test on multiple devices and browsers
